# Магазин велозапчастей "Велоспутник"

## Описание
Сайт магазина велозапчастей. Пользователи могут искать товары в каталоге, просмотривать информацию об определенных товарах,
проходить регистрацию и аутентификацию на сайте, добавлять товары в корзину, редактировать корзину, оформлять заказы, просмотаривать свой профиль.

## Ключевые особенности:
1. Разграничение прав доступа:
    - просмотр информации о товарах — для всех;
    - добавление в корзину, оформление заказов, просмотре профиля — только для аутентифицированного пользователя;
    - создание/редактирование/удаление категорий, товаров, пользователей и заказов — только для администратора.
2. Пагинация для списка выдачи поиска.
3. Поиск по названию, категории, описанию.
4. Фильтрация по категории.
5. Сортировка в прямом и обратном порядке по:
    - алфавиту;
    - цене;
    - году.
6. Валидация входных данных на уровне моделей, форм и представлений.

## Установка

### Шаги по развёртыванию

1. При необходимости установите ПО, воспользовавшись ссылками:
    - интерпритатор Python 3.12 https://www.python.org/downloads/release/python-3120/
    - эмулятор командной строки Git Bash https://git-scm.com/install/windows


2. Зайдите в директорию проекта:
```bash
cd chenk_diploma
```

3. Создайте и активируйте виртуальное окружение:
```bash
python -m venv venv
```
```bash
source venv/bin/activate  # Linux/macOS
```
```bash
source venv\Scripts\activate     # Windows
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

5. Переместитесь в папку Django-проекта:
```bash
cd cyclesat
```

6. Запустите сервер:
```bash
python manage.py runserver
```

Сайт будет доступен по адресу: http://127.0.0.1:8000/

База данных для сайта есть в проекте и уже наполнена.

В ней зарегестрирован администратор (логин: admin, пароль: admin)

Адрес админ-панели:http://127.0.0.1:8000/admin/

При необходимости создать новую базу:

1. Остановите сервер и удалите файл db.sqlite3 из корневой папки cyclesat.

Находясь в папке chenk_diploma\cyclesat:

2. Выполните миграции:
```bash
python manage.py migrate
```

3. Загрузите данные о товарах в БД:
```bash
python manage.py import_csv
```

4. Создайте аккаунт администратора:
```bash
python manage.py createsuperuser
```
Следуйте инструкциям в терминале:
    - придумайте имя
    - почту вводить необязательно
    - придумайте пароль
    - повторите пароль

5. Запустите сервер:
```bash
python manage.py runserver
```
